#Liri Bot

###Install
* Clone repo
* do an npm install


###Usage

```	
node liri.js action [terms]
	where actions: 
		concert-this
		spotify-this-song
		movie-this
		do-what-it-says
	terms: [optional for movie-this]
			 requiered for spotify-this-song and
			 		     concert-this
			 not requiered for do-what-it-says
	
			 		     
```
